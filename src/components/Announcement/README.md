# Announcement

A React component used to alert users of a particular annoncement.

# Usage

```javascript
    import { Announcement } from 'wafflecss';
    <Announcement
        data={{
            displayNode: 'This is a button'
        }}
    />
```

# Props

## Data
| Prop Name   |Is Optional    |  Type |
|----------|:-------------:|------:|
| displayNode |  false | string |

## Events

| Prop Name   |      Is Optional       |  Type |
|----------|:-------------:|------:|
| onDismiss |  true | void |
