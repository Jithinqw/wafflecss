# Loader

A React component for displaying Loader.

## Usage

A deployed version can be found [here](https://wafflecss-jithinqw.vercel.app/?path=/docs/loader--default).

## Example

```javascript
    import { Loader } from 'wafflecss';

    <Loader color="red" />
```

## Props

### Data

| Prop Name   |Is Optional    |  Type |
|----------|:-------------:|------:|
| color |  false | `string` |
| width |  false | `string` |
| height |  false | `string` |
